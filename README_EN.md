# CAN SLIM Strategy Project

## Project Overview
This is a quantitative stock screening tool based on William J. O'Neil's CAN SLIM investment strategy.

## Installation

```bash
pip install yfinance pandas numpy matplotlib
```

Or using uv:
```bash
uv pip install yfinance pandas numpy matplotlib
```

## Usage

```bash
python canslim_scanner.py
```

See README.md for detailed documentation.
